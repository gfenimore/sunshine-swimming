📋 Things you should know

Payment processing is optional — and separate

HoneyBook can collect payments from clients, but it charges 2.9% + $0.25 per card transaction (~$2 on a $60 lesson). If you'd rather keep using Venmo, Zelle, or cash for payments — which most of your clients likely prefer anyway — you skip this fee entirely. You only pay the $49/mo subscription. We'll set this up whichever way works best for you.

Free trial is 7 days — but you're covered for 67

HoneyBook's free trial is 7 days (no credit card required). But every paid plan also includes a 60-day money-back guarantee. That means you have a full 67 days to test the system risk-free — more than enough time to run it through a real swim season ramp-up. I'll have the working model built and demoed within the 7-day trial window so you can see it working before any commitment.

Annual billing saves money — but you pay year-round

The $49/mo rate requires annual billing ($588/year). Monthly billing is $59/mo and can be paused in the off-season. If you go annual, you're paying ~$245 during the Nov–Mar off-season when you're not teaching — but you can use that time for waitlist management, rebooking campaigns, and growing your digital products. We'll decide together which makes more sense once you've seen the system working.
